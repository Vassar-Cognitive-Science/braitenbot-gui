# Hardware Overview

BraitenBot is designed around a simple, low-cost two-wheeled robot that you can
build yourself. The software in this site generates Arduino sketches that drive
that robot — but you are free to adapt the wiring to whatever chassis and
sensors you have on hand.

A typical BraitenBot build consists of:

- An **Arduino-compatible microcontroller** (the brain)
- Two **continuous-rotation servos**, one per wheel, driven directly from the
  Arduino — no separate motor driver needed
- A set of **sensors** — light, distance, color, and bump sensors — mounted at
  the front
- A **chassis** that holds it all together, plus a battery pack

This section will grow into a complete do-it-yourself guide so you can build a
BraitenBot from scratch:

- **[Parts List](./bill-of-materials)** — the parts you need and where to get them
